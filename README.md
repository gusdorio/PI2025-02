# Apresentação

Projeto Integrador Segundo Semestre 2025 - Escola Politecnica PUC Campinas

## Objetivos

- Passagem dos dados amostrais para ambiente de banco de dados (respectivos a pasta 'data');
- Implementação de algoritmo responsável por predição de dados respectivos;
- Criação de interface responsável pela visualização dos resultados obtidos;


### Requisitos
- Criação de configurações docker para executar aplicação em ambiente em nuvem (Microsoft Azure);
